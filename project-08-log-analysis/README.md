# Log Analysis System

A SQL project for monitoring system logs and analyzing errors, warnings, and overall system activity.

## Features
- Store system names
- Track log entries
- Analyze errors and warnings
- Count logs by level
- Detect the most active system
- Monitor daily log activity
- Identify systems with high error rates

## Tables
- systems
- logs

## Skills Used
- SQL
- Joins
- Aggregations
- Group By
- Monitoring Analytics
- Error Analysis
- Log Reporting
