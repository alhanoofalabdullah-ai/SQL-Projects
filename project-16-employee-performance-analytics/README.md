# Employee Performance Analytics

A professional SQL project for analyzing employee productivity, attendance, and performance trends.

---

## Overview

This project demonstrates how to use SQL to evaluate employee performance based on attendance records, completed tasks, and productivity metrics.

---

## Features

- Analyze employee attendance
- Measure completed tasks
- Rank top-performing employees
- Calculate productivity metrics
- Track department performance

---

## Project Structure

- schema.sql → Database tables
- data.sql → Sample data
- queries.sql → Performance analysis queries
- README.md → Project documentation

---

## Technologies Used

- SQL
- Joins
- Aggregations
- Window Functions
- Group By
- Order By

---

## Key Insights

- Top-performing employees
- Employees with highest attendance
- Tasks completed by employee
- Department productivity
- Employee ranking by output
