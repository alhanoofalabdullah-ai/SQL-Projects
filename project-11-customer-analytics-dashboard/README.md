# Customer Analytics Dashboard

A professional SQL project for analyzing customer behavior, purchases, and revenue insights.

---

## Overview

This project demonstrates how to analyze customer activity using SQL by tracking orders, spending, and repeat purchase behavior.

---

## Features

- Identify top customers
- Calculate total customer spending
- Analyze repeat customers
- Track order frequency
- Generate customer insights

---

## Project Structure

- schema.sql → Database tables
- data.sql → Sample data
- queries.sql → SQL analysis queries
- README.md → Project documentation

---

## Technologies Used

- SQL
- Relational Database Design
- Aggregation
- Joins
- Group By
- Subqueries

---

## Key Insights

- Top spending customers
- Customers with multiple orders
- Average order value per customer
- Total revenue by customer
- Customer order count
