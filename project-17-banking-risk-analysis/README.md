# Banking Risk Analysis

A professional SQL project for analyzing banking transactions, customer balances, and identifying high-risk accounts.

---

## Overview

This project demonstrates how to use SQL to detect banking risks by analyzing customer accounts, loan balances, and unusual transaction activity.

---

## Features

- Identify high-risk customers
- Analyze large transactions
- Track loan balances
- Detect negative account balances
- Evaluate account activity patterns

---

## Project Structure

- schema.sql → Database tables
- data.sql → Sample data
- queries.sql → Risk analysis queries
- README.md → Project documentation

---

## Technologies Used

- SQL
- Joins
- Aggregations
- CASE statements
- Subqueries
- Window Functions

---

## Key Insights

- Customers with high loan exposure
- Accounts with negative balances
- Large-value transactions
- Customers with unusual transaction volume
- Risk classification by balance and loan amount
