# Hospital Database System

A SQL project for managing patients, doctors, and appointments with practical healthcare analytics.

## Features
- Store patient records
- Store doctor details and specializations
- Manage appointments
- Track appointment status (Completed, Scheduled, Cancelled)
- Analyze doctor workload
- Analyze patient activity
- Report appointments by specialization

## Tables
- patients
- doctors
- appointments

## Skills Used
- SQL
- Joins
- Aggregations
- Group By
- Order By
- Healthcare Data Analysis
