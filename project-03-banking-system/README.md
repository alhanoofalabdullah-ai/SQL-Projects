# Banking System Database

A SQL project for managing bank customers, accounts, and transactions with practical financial analysis queries.

## Features
- Store customer data
- Manage account balances
- Track deposits and withdrawals
- Analyze transactions
- Report balances and customer activity

## Tables
- customers
- accounts
- transactions

## Skills Used
- SQL
- Joins
- Aggregations
- Group By
- Order By
- Relational Database Design
- Financial Data Analysis
