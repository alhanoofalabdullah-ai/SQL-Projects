# Inventory Optimization System

A professional SQL project for monitoring stock levels, identifying fast and slow-moving items, and optimizing inventory decisions.

---

## Overview

This project demonstrates how to use SQL to analyze inventory data, detect low-stock products, measure product movement, and support reorder decisions.

---

## Features

- Detect low-stock items
- Track fast-moving products
- Identify slow-moving products
- Measure total stock value
- Generate reorder insights

---

## Project Structure

- schema.sql → Database tables
- data.sql → Sample data
- queries.sql → Inventory analysis queries
- README.md → Project documentation

---

## Technologies Used

- SQL
- Joins
- Aggregations
- CASE statements
- Group By
- Order By

---

## Key Insights

- Products below reorder threshold
- Highest-selling products
- Lowest-selling products
- Current inventory value
- Reorder recommendations
