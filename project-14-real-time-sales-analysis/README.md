# Real-Time Sales Analysis

A professional SQL project for analyzing sales performance, revenue trends, and top-selling products.

---

## Overview

This project demonstrates how to use SQL to analyze sales transactions in real time by tracking revenue, product performance, and daily sales activity.

---

## Features

- Analyze daily sales
- Track monthly revenue
- Identify top-selling products
- Measure sales by region
- Calculate average order value

---

## Project Structure

- schema.sql → Database tables
- data.sql → Sample data
- queries.sql → Sales analysis queries
- README.md → Project documentation

---

## Technologies Used

- SQL
- Joins
- Aggregations
- Group By
- Order By
- Date Functions

---

## Key Insights

- Daily revenue trends
- Monthly revenue summary
- Best-selling products

- Average order value
