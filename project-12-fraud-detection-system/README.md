# Fraud Detection System

A professional SQL project for detecting suspicious transactions and unusual activity patterns.

---

## Overview

This project demonstrates how to identify potential fraud using SQL by analyzing transaction behavior, high-value operations, and abnormal patterns.

---

## Features

- Detect high-value transactions
- Identify rapid repeated transactions
- Analyze suspicious user behavior
- Track transactions by location
- Highlight unusual activity patterns

---

## Project Structure

- schema.sql → Database tables
- data.sql → Sample data
- queries.sql → Fraud detection queries
- README.md → Project documentation

---

## Technologies Used

- SQL
- Joins
- Aggregations
- Subqueries
- Window Functions

---

## Key Insights

- Transactions above threshold
- Multiple transactions in short time
- Activity across multiple cities
- High-risk users
- Transaction frequency anomalies
