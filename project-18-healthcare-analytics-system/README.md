# Healthcare Analytics System

A professional SQL project for analyzing patient records, diagnoses, treatments, and healthcare trends.

---

## Overview

This project demonstrates how to use SQL to analyze healthcare data, track patient visits, identify common diagnoses, and evaluate treatment activity.

---

## Features

- Track patient visits
- Analyze common diagnoses
- Measure treatment frequency
- Review doctor activity
- Generate healthcare insights

---

## Project Structure

- schema.sql → Database tables
- data.sql → Sample data
- queries.sql → Healthcare analysis queries
- README.md → Project documentation

---

## Technologies Used

- SQL
- Joins
- Aggregations
- Group By
- Order By
- Window Functions

---

## Key Insights

- Most common diagnoses
- Patients with highest visit counts
- Doctor activity levels
- Treatment frequency
- Monthly visit trends
