# Advanced Reporting Engine

A SQL project for generating advanced business reports using joins, subqueries, aggregations, and views.

## Features
- Detailed sales reporting
- Revenue analysis
- Customer spending insights
- Product and category performance
- City-based order reporting
- Average order value analysis
- Reusable views for reporting

## Tables
- customers
- products
- orders
- order_items

## Skills Used
- SQL
- Joins
- Subqueries
- Aggregations
- Group By
- Order By
- Views
- Business Reporting
