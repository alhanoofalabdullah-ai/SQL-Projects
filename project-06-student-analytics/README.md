# Student Performance Analytics

A SQL project for analyzing student grades and academic performance across multiple subjects.

## Features
- Store student records
- Store subject records
- Track grades
- Calculate average score per student
- Calculate average score per subject
- Identify top and lowest students
- Filter high-performing students
- Rank students by average score

## Tables
- students
- subjects
- grades

## Skills Used
- SQL
- Joins
- Aggregations
- Group By
- Having
- Order By
- Academic Performance Analysis
