# Inventory Management Database

A SQL project for managing products, suppliers, and stock movements with practical inventory analysis queries.

## Features
- Store supplier data
- Manage products and categories
- Track stock quantities
- Record stock in and stock out movements
- Analyze low stock products
- Calculate inventory value
- Report supplier and category performance

## Tables
- suppliers
- products
- stock_movements

## Skills Used
- SQL
- Joins
- Aggregations
- Group By
- Order By
- Inventory Analysis
- Relational Database Design
