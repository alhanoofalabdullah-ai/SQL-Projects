# Sales Database

A SQL project for managing customers, products, and orders, with practical sales analytics queries.

## Features
- Store customer data
- Store product catalog
- Track orders
- Calculate total revenue
- Analyze sales by product
- Analyze sales by customer
- Filter sales by city and category

## Tables
- customers
- products
- orders

## Skills Used
- SQL
- Joins
- Aggregations
- Group By
- Order By
- Subqueries
- Relational Database Design
