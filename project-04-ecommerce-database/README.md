# E-Commerce Database

A SQL project for managing customers, products, orders, and order items in an online store.

## Features
- Store customer data
- Manage product catalog
- Track orders and order items
- Calculate total revenue
- Analyze best-selling products
- Analyze top customers
- Report stock and category revenue

## Tables
- users
- products
- orders
- order_items

## Skills Used
- SQL
- Joins
- Aggregations
- Group By
- Order By
- Relational Database Design
- Sales and Revenue Analysis
