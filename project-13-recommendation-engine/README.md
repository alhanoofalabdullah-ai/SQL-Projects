# Recommendation Engine

A professional SQL project for generating product recommendations based on customer purchase behavior.

---

## Overview

This project demonstrates how to use SQL to recommend products by analyzing customer orders and finding products frequently bought together.

---

## Features

- Identify products bought together
- Generate cross-sell recommendations
- Analyze customer purchase patterns
- Rank product combinations
- Build recommendation logic using SQL

---

## Project Structure

- schema.sql → Database tables
- data.sql → Sample data
- queries.sql → Recommendation queries
- README.md → Project documentation

---

## Technologies Used

- SQL
- Joins
- Aggregations
- Self Joins
- Group By
- Order By

---

## Key Insights

- Products frequently purchased together
- Most popular products
- Customer purchase combinations
- Product pair ranking
- Cross-sell opportunities
