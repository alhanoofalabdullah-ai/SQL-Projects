# Log Monitoring System

A professional SQL project for analyzing system logs, detecting frequent errors, and monitoring application events.

---

## Overview

This project demonstrates how to use SQL to store and analyze log data, detect system failures, and identify recurring issues over time.

---

## Features

- Track log events
- Detect frequent errors
- Analyze log levels
- Monitor events by date
- Identify affected systems

---

## Project Structure

- schema.sql → Database tables
- data.sql → Sample data
- queries.sql → Log monitoring queries
- README.md → Project documentation

---

## Technologies Used

- SQL
- Joins
- Aggregations
- Group By
- Order By
- Date Functions

---

## Key Insights

- Most frequent error messages
- Log counts by level
- Daily event volume
- Systems with highest error count
- Critical issue tracking
